# Ракурсы ботинка для страницы Advantages

Сюда помещаются фото ракурсов ботинка для страницы «Преимущества» (`/advantages`).

**Структура:**
- `2sk/black/1.png` … `5.png` — ракурсы, цвет черный
- `2sk/oliva/1.png`, `1_2.png`, `1_3.png`, `2.png`, `3.png`, `4.png`, `5.png`, `6.png` и т.д. — ракурсы, цвет олива (имена файлов как в текущей папке `views/models/2sk/oliva/`)

Скопируйте или переместите содержимое `views/models/2sk/` в эту папку (внутри должна быть папка `2sk` с подпапками `black` и `oliva`).
